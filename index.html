Portfolio

Services

We focus on creating and managing business solutions using Oracle technology, the operations that surround it, and the Oracle service infrastructure necessary to achieve the highest levels of service (e.g., availability, performance, etc.) and customer satisfaction.

Database migration - Upgrade

We carry out the complete migration activity, from the total planning of the process, to the post migration support. We provide support for any type of upgrade or migration from oracle 11g to 19c. We work with technologies to ensure "zero-dowtime" in migration processes.

Database deployment

We install, configure and provide support for any type of database or grid infrastructure deployment from 11g to 19c, both on dedicated servers such as EXADATA or Oracle Cloud Infrastructure. Including technologies: GRID, RAC, DataGuard, ASM, GoldenGate.

Database administration

We provide advanced troubleshooting and administration support for any type of database or grid infrastructure deployment from 11g to 19c, both on dedicated servers such as EXADATA or Oracle Cloud Infrastructure. Including technologies: GRID, RAC, DataGuard, ASM, GoldenGate.

Oracle Cloud Infrastructure (OCI)

We have extensive experience in the deployment and administration of Oracle Cloud Infrastructure (OCI). From the VM deployment, to the complete configuration of EXACS (Exadata Cloud Service), and the management of the stack and its components.

GoldenGate

We provide support for the installation, configuration, and administration of Oracle GoldenGate, within any supported OS, on any database as of 11g. We have a great experience with troubleshooting of this technology, and its different types of configurations.
